Augen ↔ Figma

Pull an Augen creative onto the canvas, edit its copy & layout, push it back.

First time? Setup (3 steps)
  1. In Augen, open Settings → MCP & API and click Generate token. Copy it.
  2. Paste your Augen URL + that token below, and the brand slug (the part after /brands/ in the URL).
  3. Load creatives → Import one → edit the augen:* text layers → select the frame → Send selection → Augen.
Edit: change the augen:* text layers (and move the headline / CTA) on an imported frame, select it, then Send selection → Augen. The render image is a locked backdrop — copy + layout sync back; re-render in Augen to see it.
